# Rolox

**Versão:** 1.0.0  
**Criador Principal:** Roblox Official  
**Publicador:** One Mapas Official  
**Plataforma:** Windows  
**Framework:** .NET 8 / WinForms  

---

## O que é o Rolox?

Rolox é um cliente alternativo do Roblox para Windows, desenvolvido pela One Mapas Official.  
Ele permite jogar, explorar jogos populares, gerenciar amizades e hospedar servidores privados gratuitos diretamente do seu PC.

---

## Funcionalidades

- Login com conta Roblox via Edge
- Sessão salva automaticamente (sem precisar logar toda vez)
- Home com jogos populares, recomendações e continuar jogando
- Aba Destaques com jogos em destaque, revelações e mais jogadas (beta)
- Aba Turma com lista de amigos e acesso a comunidades (beta)
- Aba Mais com acesso rápido a Mercado, Premium, Inventário, etc. (não testado)
- Servidores privados P2P gratuitos (roda no PC do host) (não testado)
- Chat com moderação automática por IA (ban progressivo) (não testado)
- Busca de jogadores pelo username (beta e nãotestado)
- Perfil de outros jogadores com botão Join se estiverem hospedando servidor (não testado)
